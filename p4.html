IDEAL

The problem present two strings that need to represent the same word by removing characters, but checking the size by their ASCII representation.

The main idea here is to recursively get the size of each word using their ASCII code and fill and array. After compare the string S1 and S2 to check which of them is shorter. Let state S1 is the shortest, then compare which words have in common with S2. Then check the size of both words in common from S2 and S1 then remove them from the word. Save them into an array and finally calculate their weight. Also checking if the words with characters are already removed are the same.

Recursive definition could be set as: Short word (get min) dp(S) = dp[i] < dp[i+1] get sum dp(s) = dp+s, check values are same, dp(s,sp)= dp[i]==dp[sp],

In conclusion we use a small solution to apply to a big problem so it can be reused as many times as possible
