Minimum Falling Path Sum

Given a square array of integers A, we want the minimum sum of a falling path through A.

A falling path starts at any element in the first row, and chooses one element from each row. The next row's choice must be in a column that is different from the previous row's column by at most one.

Code
                            
                                    class Solution(object):
                                    def minFallingPathSum(self, A):
                                        while len(A) >= 2:
                                            row = A.pop()            
                                            for i in xrange(len(row)):
                                                A[-1][i] += min(row[max(0,i-1): min(len(row), i+2)])
                                        return min(A[0])
IDEAL

The idea in this problem can be solved using dynamic program since from a small portion of the problem everyelse can be solved in a large scale. Each of the paths contain some amount of numbers that in a sum have to be the minimum path.

TThe approach to take while using dynamic program is basically getting each element of an array then comparing it with other elements in the subarray, and added to another array (final path). Then do on to the next one get the minimum value then added again to the minimum array, until it reaches the end of the array parent and no more subarrays. Then sum all the values in the array and finally get the sum which will result in the minimum.

Recursive definition could be set as: dp(r, c) which is A[r][c] + min(dp(r+1, c-1), dp(r+1, c), dp(r+1, c+1))
