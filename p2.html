Palindromic Substrings

Given a string, your task is to count how many palindromic substrings in this string.

The substrings with different start indexes or end indexes are counted as different substrings even they consist of same characters.

Code
                            
                                class Solution(object):
                                    def countSubstrings(self, S):
                                        N = len(S)
                                        ans = 0
                                        for center in xrange(2*N - 1):
                                            left = center / 2
                                            right = left + center % 2
                                            while left >= 0 and right < N and S[left] == S[right]:
                                                ans += 1
                                                left -= 1
                                                right += 1
                                        return ans
IDEAL

This problem can be solved by tearing the problem apart in small section. A palindrome is basically a word that can be read same backwards. But this time in a single string how many substrings are palindromes.

The idea to solve this problem can is recursive/iterative by doing an iteration on the string. Then using two pivots that mark from point to point. This helps by standing in a character then checking the rest of them to check if they are palindrome. In code: S is String so, the idea is S[i] pivot 1 then S2[i+1] then iterate S2 so that reach until the end of the String. If S2 finds same character and the next one is not the same as S[i] then stops. If finds same character then add to an array the word backwards and all the posibilities with them

Recursive definition could be as: n as length of String, S[i], S[i+1]...[Sn-1]
